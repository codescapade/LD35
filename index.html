Squishy's Escape

Ludum Dare 35 Compo Entry

View on GitHub

Squishy's Escape is a 2d Platformer. The Ludum Dare theme was Shapeshift so squishy can change its shape to hekp get through the levels. I ended up making only 6 levels because I ran out of time so it isn't a very long game.

You have to click inside the game window for the keys to work

Controls

Arrow keys to move.
Spacebar to Jump.
Press and Hold Spacebar to float while in the air.
Hold the down arrow to turn into a block to destroy broken tiles.
